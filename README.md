## 🚀 Deploy on Vercel

### Step 1 — GitHub pe upload karo

```bash
# New repo banao: pandit-pkp
# index.html naam se file upload karo
```

### Step 2 — Vercel se connect karo

1. [vercel.com](https://vercel.com) pe jao
2. GitHub se login karo
3. **"Add New Project"** → `pandit-pkp` repo select karo
4. **"Deploy"** click karo — bas! ✅

### Step 3 — Live URL

```
https://pandit-pkp.vercel.app
```
